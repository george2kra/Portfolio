Inventory Management

The Inventory program manages the placement of bulk goods by locating an empty shelf that fits the height and weight of the pallet. The details are recorded and placed into an array for later retrieval.

With in the program can define new rack locations, the shelves height and weight limitations.

Further detials can be added such as secure locations, refrigeration, or shelves for storing dangerous goods.

Rainfall collection

A coding challenge to Collect rainfall details for any place on any month of the year.

The program will prompt the user to enter the details onto the screen and the data will be stored into a CSV file, the user is then able to invoke a spread sheeting graphing tool and compare rainfall differences between towns.
